LunaTV 配置中心 V1 架构

目标：让 LunaTV 做统一内容配置后台，OrionTV 只负责拉取并消费配置

核心数据流
中心后台

LunaTV Admin

  • 点播源 SourceConfig
  • 直播源 LiveConfig
  • 分类 CustomCategories
  • 公告 Announcement
复用现有 admin 页面与存储能力
→
适配层

Orion Config API

  • 读取 LunaTV AdminConfig
  • 裁剪 V1 允许字段
  • 转换为 TV 稳定结构
  • 输出版本号 / 更新时间
推荐新增独立接口，不让 TV 端直接吃后台内部模型
→
TV 客户端

OrionTV

  • 启动时拉取配置
  • 本地缓存配置快照
  • 源列表 / 直播页消费
  • 设置页只展示内容配置结果
V1 不接用户权限、不接设备中心
后台负责
  • 统一维护点播源与直播源
  • 统一维护分类入口
  • 统一发布公告
  • 提供给 TV 的只读配置接口
TV 端负责
  • 拉取并缓存内容配置
  • 按配置渲染搜索 / 首页 / 直播
  • 保留本地播放与遥控逻辑
  • 配置拉取失败时回退本地缓存
